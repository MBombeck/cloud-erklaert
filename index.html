Was steckt eigentlich in einem Computer?

Jeder Computer besteht aus diesen 3 Grundbausteinen

Ein Computer, eine Aufgabe

Frueher hatte jeder Server genau einen Job

3 Aufgaben = 3 physische Server! Viel Hardware, wenig Auslastung.

Der Serverraum

Viele Server brauchen viel Platz, Strom und Kuehlung

Hohe Kosten, viel Aufwand, wenig Flexibilitaet

Virtualisierung

Ab ca. 2000: Ein physischer Server, viele virtuelle Maschinen

Wie ein Mehrfamilienhaus: Ein Gebaeude, viele Wohnungen
Bessere Auslastung, weniger Hardware!

Moderne Rechenzentren

Riesige Rechenzentren der Cloud-Anbieter

Statt eigene Server zu kaufen, mietet man Rechenleistung bei Profis

Infrastructure as a Service

IaaS — Infrastruktur mieten

Wie ein Mietwagen: Sie fahren, aber kuemmern sich nicht um die Werkstatt
Virtuelle Maschinen auf Knopfdruck

Platform as a Service

PaaS — Plattformen mieten

Wie eine moeblierte Wohnung: Kueche ist fertig, Sie muessen nur noch kochen!
Weniger Verwaltungsaufwand, mehr Fokus auf Fachlichkeit

Software as a Service

SaaS — Fertige Software nutzen

Wie ein Restaurant: Sie bestellen das fertige Essen. Kein Kochen, kein Abwaschen!
Einfach im Browser oeffnen und loslegen
